# GIT

### 1. git clone 代理报错
```
git clone https://github.com/ly2010dd/Sparrow.git
Cloning into 'Sparrow'...
fatal: unable to access 'https://github.com/ly2010dd/Sparrow.git/': Could not resolve proxy: http; Unknown error
```
- 检查.gitconfig文件中代理设置
- 检查本地环境变量中是否设置代理

```
echo $http_proxy
echo $https_proxy

unset $http_proxy
unset $http_proxy
```
